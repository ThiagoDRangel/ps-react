# Quinto Desafio - React

Este é o quinto desafio do projeto, que consiste em utilizar o React para criar uma aplicação web. O objetivo é exibir informações sobre transferências em uma página.

## Como Utilizar o Projeto

Siga as instruções abaixo para utilizar o projeto: </br></br>

> Clone esse projeto `git clone git@github.com:ThiagoDRangel/ps-react.git` </br></br>
> Instale as dependências `npm install` </br></br>
> Inicie o servidor React com `npm start` </br></br>

- O frontend `React` criará uma tela para exibir as informações de transferências, utilizando os dados fornecidos pelo servidor `Java`.

- O Servidor `Java` se encontra disponível abaixo:
  ```bash
  git clone git@github.com:ThiagoDRangel/PS-Java-React.git
  ```

OBS: O Frontend espera uma conexão com a porta 8080
